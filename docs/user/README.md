# User Guides

[Documentation Home](../README.md)

These pages collect the user-facing information from the project wiki and
cross-check it against this repository where possible. Wiki material is treated
as external project documentation: when a statement appears historical,
version-specific, or not verified in this checkout, the page says so.

## Start Here

- [Quick Start](quick-start.md): common conversion recipes.
- [Prebuilt Packages And Docker](../build/prebuilt-packages.md): package and
  container options described by the wiki.
- [Optimization And Customization](optimization-and-customization.md): manifest,
  embedding, static assets, PDF pre-processing, and output-size tips.
- [Browser And Mobile Notes](browser-and-mobile.md): browser feature
  requirements and mobile caveats.
- [FAQ And Limitations](faq-limitations.md): common user problems, known
  limitations, and deliberately unsupported feature areas.
